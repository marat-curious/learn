### Введение

Компиляторы:
- GNU Compiler Collection (GCC)
- Clang
- Intel C++ Compiler

При проверке задач мы используем следующие ключи компилятора:

```
clang++ --std=c++20 -O3 -fsanitize=address,undefined -Wall -Wextra -Werror
```
